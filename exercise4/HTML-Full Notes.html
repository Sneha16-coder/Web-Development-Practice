What is HTML?

HTML is the Standard Markup Language for creating web sites or web pages.
HTML Stands for Hyper Text Markup Language.
HTML describes the structure of a web page.
HTML is developed by Tim Berner's Lee
HTML consists of a series of Elements.

Basic Structure of HTML:

Let's know the Structure of Basic HTML


<!DOCTYPE HTML>
<html>
<head>
<title> Write Here Page Title </title>
</head>
<body>
<h1> This is a Heading </h1>
<p> This is a Paragtaph </p>
</body>
</html>

Now let us know What is meant by Element.

What is Element?

An HTML Element is defined by a start tag, some content and End tag. The HTML Element is everything from start tag to end tag.

<tagname> Content goes here </tagname> and this is basically all elements are written in this format

For Example : <h1>First Heading</h1>

Note :

The content inside the <body> section will be displayed in a browser .
The content inside the <title> element will be shown in the browser's title bar.

Now We will introduce you a New Term that is called Attribute

What is HTML Attribute?

HTML Attribute provides additional information about HTML elements.
Attributes are always specified in the start tag
Attributes Usually come in pairs like name="value"

For Example :
href attribute :
<a href="https://google.com>Go to Google</a>
src attribute :
<img src="sneha.jpg">
width, height and alt attribute :
<img src="h1.jpg width="25px" height="25px" alt="Home pic">
title attribute :
<p title="I am a tooltip">Sneha</p>

Table Tags:

Table consists of table cells inside rows and column.
Some Different Table Tags we will discuss as Following:

Table Cells : Everything between <td> and </td> are the content of the table cell.
Table Rows : Each Table row starts with a <tr> and ends with </tr> tag.
Table Headers : Sometimes you want your cells to be table header cells. in those cases use the <th> tag instead of <td> th stands for table header.
Colspan : To make a cell span over multiple columns, use the colspan attribute
Rowspan : To make a cell span over multiple rows, use the rowspan attribute

ShortNotes on Table Tags
<table> : defines a table
<th> : defines a header cell in a table
<tr> : defines row in a table
<tde> : defines a cell in a table
<caption> : defines a table caption
<thead> : Groups the header content in a table
<tbody> : Groups the body content in a table
<tfoot> : Groups the footer content in a table
